📄 PDF Editor

How to use:

  • Load PDF: Click "Choose File" and select a PDF
  • Edit Text: Click "Edit Text" - editable text boxes will appear over all text. Type directly in the boxes like Google Docs!
  • Add New Text: Click "Add Text", enter text, then click anywhere on the PDF
  • Add Signature: Click "Signature", enter your name, then click on the PDF
  • Navigate: Use Previous/Next buttons to move between pages
  • Save: Click "Save PDF" to download your edited PDF

Note: Backend server (Python + PyMuPDF) must be running for proper font preservation. Start it with: cd backend && ./start.sh
